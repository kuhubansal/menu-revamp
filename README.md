# menu-revamp

Goal: Revamp the site menu management UI + backend. Backend: Node.js + Express + TypeScript + SQLite (better-sqlite3). Frontend: to be created (React recommended).

This repo has two folders:
- backend/   (Express + TS)
- frontend/  (empty for now)

Steps to run backend (from backend/):
1. npm install
2. npm run dev
3. open http://localhost:3000/api/health
